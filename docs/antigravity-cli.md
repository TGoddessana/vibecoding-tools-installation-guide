## 설치 방법

### 1. Python 버전 확인

```bash
python3 --version
# Python 3.10.x 이상이어야 합니다
```

### 2. Antigravity CLI 설치

```bash
pip install antigravity
# 또는
pip3 install antigravity
```

{: .tip }
가상환경(`venv`)을 사용하면 의존성 충돌을 방지할 수 있습니다.

### 3. 설치 확인

```bash
antigravity --version
```

### 4. 실행

```bash
antigravity
```
