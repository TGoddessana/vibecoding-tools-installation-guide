시스템 요구사항

항목요구사항
Node.js22 이상

OSmacOS, Linux, Windows (WSL 권장)
OpenAI API Key필요 — 발급 페이지

설치 방법

  1. Node.js 설치 확인 Node.js 22 이상이 설치되어 있는지 확인합니다. 
    node --version
# v22.x.x 이상이어야 합니다

npm --versionshell
    💡

    Node.js가 없다면 nodejs.org 에서 LTS 버전을 설치하거나, nvm을 사용하세요.

  2. Codex CLI 전역 설치 npm으로 @openai/codex 패키지를 전역 설치합니다. 
    npm install -g @openai/codexshell
  3. API Key 환경변수 설정 OpenAI API Key를 환경변수로 등록합니다. 
    # ~/.zshrc 또는 ~/.bashrc에 추가
export OPENAI_API_KEY="sk-..."shell
    # 즉시 적용
source ~/.zshrcshell
    ⚠️

    API Key는 절대 코드에 직접 넣지 마세요. 환경변수 또는 .env 파일을 사용하고, .gitignore에 추가하세요.

  4. 설치 확인 및 실행 설치가 완료되면 버전을 확인하고 실행해봅니다. 
    codex --version

# 실행
codexshell
