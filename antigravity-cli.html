🚀 Antigravity CLI
터미널에서 동작하는 AI 코딩 에이전트 — 자연어로 복잡한 개발 작업을 처리합니다.

설치 영상을 여기에 추가하세요 YouTube embed 또는 <video> 태그로 교체

시스템 요구사항

항목요구사항
Python3.10 이상
pip최신 버전 권장
OSmacOS, Linux, Windows (WSL 권장)

설치 방법

  1. Python 버전 확인 Python 3.10 이상이 설치되어 있는지 확인합니다. 
    python3 --version
# Python 3.10.x 이상이어야 합니다shell
  2. Antigravity CLI 설치 pip으로 패키지를 설치합니다. 
    pip install antigravity
# 또는
pip3 install antigravityshell
    💡

    가상환경(venv)을 사용하면 의존성 충돌을 방지할 수 있습니다.

  3. 설치 확인 설치가 완료되면 버전을 확인합니다. 
    antigravity --versionshell
  4. 실행 터미널에서 바로 실행해봅니다. 
    antigravityshell
⚠️

설치 정보가 업데이트될 수 있습니다. 최신 내용은 공식 문서를 반드시 확인하세요.
