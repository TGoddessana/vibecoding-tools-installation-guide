# 바이브코딩 툴 설치 가이드

비전공자와 비개발자도 설치 결정을 쉽게 할 수 있도록 만든 실전 가이드입니다.  
"어떤 툴부터 깔아야 하지?"라는 질문에 바로 답할 수 있게, 용도별로 시작점을 정리했습니다.

---

## 구독제별 코딩 툴

| 당신의 상황 | 먼저 설치할 툴 | 이유 |
|---|---|---|
| 이미 Claude 유료 구독 중 | [Claude Code]({% link docs/claude-code.md %}) | 추가 결제 없이 가장 빠르게 시작 가능 |
| OpenAI API를 이미 사용 중 | [Codex CLI]({% link docs/codex-cli.md %}) | 기존 OpenAI 워크플로우와 바로 연결 가능 |
| 새로운 도구를 비교 검토 중 | [Antigravity CLI]({% link docs/antigravity-cli.md %}) | 설치 후 작업 방식과 비용 구조를 직접 비교하기 좋음 |

---

## 만약 당신이 클로드 구독자라면

**Claude Code부터 시작하는 것이 가장 합리적입니다.**  
이미 구독 중이라면 비용과 설정 부담이 낮고, "설치 후 바로 써보기"까지의 거리가 짧습니다.

권장 순서:
1. [Claude Code 설치 가이드]({% link docs/claude-code.md %})를 먼저 진행합니다.
2. 실제로 한 번 사용해 본 뒤, 필요한 작업 유형을 확인합니다.
3. 더 자동화가 필요하면 [Antigravity CLI]({% link docs/antigravity-cli.md %})를 추가합니다.
4. OpenAI 워크플로우가 필요할 때 [Codex CLI]({% link docs/codex-cli.md %})를 확장 도입합니다.

---

## 비전공자/비개발자를 위한 의사결정 기준

| 기준 | 체크 질문 | 추천 |
|---|---|---|
| 비용 효율 | "이미 결제한 서비스가 있나?" | 있으면 그 생태계 툴부터 시작 |
| 초기 난이도 | "터미널이 낯선가?" | 설치 단계가 짧은 가이드부터 시작 |
| 목표 작업 | "내가 하려는 건 문서/자동화/코드 중 무엇인가?" | 목표와 가장 가까운 툴 1개만 먼저 도입 |
| 확장성 | "나중에 다른 툴도 쓸 계획이 있나?" | 첫 툴 적응 후 2번째 툴을 추가 |

실수 줄이는 원칙:
1. 한 번에 하나의 툴만 설치한다.
2. 설치 직후 "작동 확인"까지 끝낸다.
3. 다음 툴은 첫 툴이 익숙해진 뒤에 추가한다.

## 이 가이드에 대해

이 페이지는 바이브코딩에 사용되는 AI 툴을 직접 설치하며 기록한 실사용 가이드입니다.  
각 문서는 "준비물 → 설치 → 실행 확인" 흐름으로 따라갈 수 있게 정리했습니다.

{: .note }
각 가이드는 macOS / Linux 환경을 기준으로 작성되었습니다. Windows 환경에서는 일부 명령어가 다를 수 있습니다.
