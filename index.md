# 바이브코딩 툴 설치 가이드

AI 코딩 어시스턴트 툴들을 직접 설치하면서 기록한 가이드 모음입니다.  
각 툴의 설치 과정을 영상과 함께 단계별로 정리했습니다.

---

## 가이드 목록

| 툴 | 설명 |
|---|---|
| [📦 Codex CLI]({% link docs/codex-cli.md %}) | OpenAI의 터미널 기반 AI 코딩 어시스턴트 |
| [🚀 Antigravity CLI]({% link docs/antigravity-cli.md %}) | 자연어로 동작하는 터미널 AI 코딩 에이전트 |
| [🤖 Claude Code]({% link docs/claude-code.md %}) | Anthropic Claude를 터미널에서 직접 사용하는 CLI |

---

## 이 가이드에 대해

이 페이지는 바이브코딩에 사용되는 AI 툴들을 **직접 설치해보면서** 기록한 개인 설치 가이드입니다.  
각 툴마다 설치 영상과 함께 단계별 명령어를 정리해두었습니다.

{: .note }
각 가이드는 macOS / Linux 환경을 기준으로 작성되었습니다. Windows 환경에서는 일부 명령어가 다를 수 있습니다.
